Expand description

Write GridMap2D comprised of tiles containing VisTileData into provided ImageBuffer. Pixel data retrieved via VisTileData::vis_pixels will be used.

To make sure that the image buffer has exact correct size needed to write the GridMap2D representation into it, you can use check_grid_image_size beforehand.
